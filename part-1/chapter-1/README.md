- applet 
- 企业级 JavaBean（Enterprise JavaBean，EJB）
- Java数据对象（Java Data Object，JDO）

以及无数的日志框架，但是作为一个平台，Java的历史是丰富多彩的，有很多企业级软件都是基于这个平台构建的。Spring 是 Java 历史中很重要的组成部分。
